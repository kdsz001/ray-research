## 核心结论

- **基本面顶级，代币反人性**：保护加密世界 $327 亿 TVS、占赛道 58.6%；但 LINK 距 ATH 跌 80%、2025 全年 -33%（同期 BTC -3%）
- **结构性卖压**：273M LINK 在 Foundation 钱包，无 vesting 合约，2022-2026 累计向 Binance 转入 88.95M LINK (~$805M)
- **机构端真有进展**：SWIFT、UBS uMINT、Coinbase Wrapped、Sony Soneium 是 production 级别；但 BlackRock BUIDL 不用 Chainlink（普遍混淆）
- **投资定性**：蓝筹基础设施仓位，不是财富密码。配置 15-25%、分批建仓、解锁后 1 周买入是推荐路径
