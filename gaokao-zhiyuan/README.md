## 核心结论

- **盘子是亿元级不是百亿** — 付费市场约 ¥10 亿（艾媒单一口径），增速降到个位数，"近九成愿意"但真实付费率推算仅 4–5%
- **头部真空** — 张雪峰 2026-03-24 因心源性猝死去世（7 家一手媒体已交叉验证），撑起赛道的"个人信任背书"出现裂缝
- **工具层已被大厂免费国有化** — 夸克/百度/腾讯 AI 志愿助手全免费 + 教育部"阳光志愿"官方系统，新人别在工具层卷
- **政策风险中高** — 教育部连续 3 年专项治理 + 广告法第 24 条 + 抖音健康分封号机制，剑指"违规营销话术"
- **新人唯一的缝** — 某省 + 中低分段 + AI 出底稿 + 人工补决策和情绪陪跑；第一年蓄水，2027 才收割
