## 核心结论

- **基本面胜利、定价失败** — Forward PE 7.88 / EV/EBITDA 5.64 / 净现金占市值 44% / 净利率 22.7%，所有指标都说"应该值更多"；但市场不肯给——因为没港股兜底 + 不分红不回购 + 2026-04 Co-CEO 个人被罚"暴力抗法" + AI 缺位四张牌还在桌上
- **赌的不是基本面，赌的是管理层** — 基本面已经够好。赌的是"管理层是否愿意把账上 $60B 现金部分还给股东、或证明再投资能产生回报"
- **三道刀同时悬挂** — 美国关税（已落）+ 国内监管（已落）+ AI 缺位（持续）。前两道已大致定价；AI 缺位是结构性折价，短期无反转
- **不要单一仓位超过流动性资产 10%** — HFCAA 强制退市是真实尾部风险，PDD 没有港股兜底，粉单市场流动性极差
