## 核心结论

- **方法论内核** — 从 AI 必然的物理趋势出发，沿供应链逐层倒推，找"小到没人看、却谁都绕不开"的最窄卡脖子环节（Chokepoint Theory），在机构与财报反应前埋伏小盘股
- **学他的"问法"，别抄他的"答案"** — 提问与拆解框架是真本事、可迁移；但具体喊单不可抄，等你听说时最肥的 9–11 倍已走完
- **业绩存疑** — 自报「2 年 225 倍」收益被 X 社区笔记标"误导性"；可验证的只有"他点过的票后来涨了"（TradingKey 回测胜率约 86%），个人账户收益无法核实
- **两个大裂缝** — ①框架只讲买、不讲卖（无估值/卖出纪律，本人承认）；②言行落差：嘴上"上游小盘卡脖子"，手上最大仓位漂到热门算力股 $NBIS
- **重大更正** — 二手提炼流传的"反期权铁律"系误读，他本人推文显示重度使用期权（leaps/covered call）与杠杆
