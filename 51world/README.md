## 核心结论

- **朋友推荐方向对，但标的有偏差**——五一视界是中国数字孪生 + 物理 AI 叙事里最靠前的上市标的，但<strong>不是布局物理 AI 的核心通道</strong>。
- **"NVIDIA Global L4 Simulation Partner" 是单方面 PR 头衔**——NVIDIA 官网 site:nvidia.com / blogs.nvidia.com / developer.nvidia.com 三处搜索全部零结果，PR 稿无 NVIDIA 高管原话。真实身份：Inception 成员 + NuRec 整合方之一（与 dSPACE、Foretellix 并列）。
- **51WorldModel 没有 arxiv 论文 / 没有开源代码 / 没有第三方评测**——是为 IPO 故事打造的 PR 产品，技术底子不如 NVIDIA Cosmos / 商汤开悟。
- **真壁垒在 L3 智驾仿真（53.5% 国内市占）**——六大国家级检测机构 100% 覆盖 + 2026 准入新规把仿真验证写成强制项。这是它能赚到的真饭。
- **财务画像很难看**——3.5 年累亏 ¥4.88 亿、毛利率从 65% 跌到 41%、研发人员从 250 砍到 114、客户从 183 跌到 131 家、51Sim 留存率仅 22.4%、应收账款周转 522 天。
- **2024 年最大单一客户是北京昌平区智慧村项目**（占收入 14.8%），前五大客户没有一家车企——和"物理 AI"几乎无关。
- **决策建议三选一**：Path A 分散下注 L4 本体（宇树/智元/银河通用 + 极佳视界）｜Path B 把五一视界当渠道用｜Path C 谨慎买入 6651.HK 股票。
