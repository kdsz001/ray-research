01 · 一页纸结论

5 个调研员, 一个结论

派出去的 5 个并行子 Agent (海外英文 / 中文圈 / HL 关系 / 团队融资 / 空投策略) 都收回数据, 关于 trade.xyz 的真相高度收敛, 没有大的内部分歧。要点如下:

它不是普通的 Hyperliquid 前端。Builder Code (HL 允许第三方做前端抽手续费的机制, 比如 PVP.trade、Phantom 走的就是这条路) 是套壳玩法。trade.xyz 走的是更深的 HIP-3 — Hyperliquid Improvement Proposal 3, 一个允许人押 50 万枚 HYPE (约 $20-25M USD) 后, 在 Hyperliquid 链上自己开一个新永续市场的协议。trade.xyz 是 HIP-3 上线首日就部署的第一个项目, 而且占了整个 HIP-3 90% 以上的市场份额。

建造它的是 Hyperunit — Hyperliquid 的资产 tokenization (代币化) 姐妹团队。法律上 trade.xyz 自己是 XYZ Ltd, 团队完全匿名; 但行业默认它是 Hyperliquid 的"长子项目"。手续费 50/50 分给 Hyperliquid 协议金库和 trade.xyz, 这意味着 trade.xyz 跑得越好, HYPE 通过协议费回购销毁吃得越多。

如果你今天就要做决定, 记住这三件事

  1. 没有官方 Points, 但 USDC "事实空投" 在发。 官网 docs、Twitter、Telegram 全部没有提到 Points / Season / Airdrop。但 DefiLlama 的财报显示 Q2 2026 trade.xyz 已经发了 $1.55M USDC 的推荐返佣 (Referral & Trading Discounts) — 这就是事实空投, 只不过给的是 USDC 不是代币。
  2. 双重撸毛是最大价值。在 trade.xyz 上交易, 同时累计 $HYPE Season 2 权重 + 可能的 $UNIT (Hyperunit 母品牌) 空投权重。"一份资金两份票"是这次调研最关键的撸毛 insight。
  3. 不要工业化刷量。Arrakis 4 月报告显示 79K 钱包里 44% 是女巫, 而且 99.9% 来自同一个 Polymarket 老炮 "Themino" 的农场 — 那个人烧了 $34,510 USD 协议费搭起 3.4 万小号, 现在大概率会被一刀切。要撸有效, 必须做真用户形态: $5K-$50K 资金、多种资产、多时段、独立资金链路。

剩下要看的是细节: 它在做什么标的、监管会不会动它、KOL 怎么看、中国人能不能用、应该投多少资金。下面 12 章逐条拆。

03 · 与 Hyperliquid 关系

不是套壳前端, 是亲儿子级 deployer

用户最关心的问题之一: trade.xyz 到底是不是 Hyperliquid 的官方产品? 是同一团队吗? 还是套壳?

简短回答: 法律上独立 (XYZ Ltd vs Hyperliquid Labs), 技术上深度绑定 (跑在 HL 链上, 共享撮合引擎, 分一半手续费), 团队上是姐妹团队 (Hyperunit)。不是 Builder Code 套壳, 是 HIP-3 部署 — 这两者的区别决定了 trade.xyz 的护城河。

三种可能性 vs 真相

假设对不对说明
是 Hyperliquid Labs 100% 子公司 半对 不是直接子公司, 但Hyperunit 团队和 Hyperliquid Labs 共列 "Guardian Set", 资金和人深度同源。
用了 Builder Code 的第三方前端 错 Builder Code 是 PVP.trade、Phantom 走的路 — 只做前端, 抽前端手续费; trade.xyz 走的是更底层的 HIP-3, 自己定义市场, 收 50% 手续费分成。
独立竞争对手 错 不是竞争, 是 Hyperliquid 用 trade.xyz 拓展"美股+大宗+外汇"这条第二增长曲线。trade.xyz 越成功, HYPE 通过协议费回购越多。
流动性聚合器 错 流动性只来自 Hyperliquid 撮合引擎, 不聚合任何其他 DEX。
部署在 EVM / Solana 错 直接跑在 Hyperliquid L1 的 HyperCore 上, 不是 HyperEVM 也不是任何外链。

监管事件的真实威胁度

为什么这件事值得关注

CME 和 ICE 是美国传统衍生品市场的两大支柱 (CME = 芝商所、ICE = 纽交所母公司), 它们联合施压 CFTC (美国商品期货交易委员会) 表明传统金融已经认真把 Hyperliquid 当威胁。trade.xyz 的 CL (原油) perp 和 SPCX (SpaceX) 这种 Pre-IPO 永续, 在 SEC / CFTC 眼里都是未注册证券/商品衍生品。

已 mitigation: trade.xyz 软地理屏蔽 US/CA/UK, Hyperliquid 已设 Hyperliquid Policy Center 公开应对。但"CFTC 立案 + 制裁 Hyperliquid 美国节点"是真实可能性。

对撸毛玩家含义: 监管事件可能加速 trade.xyz 发 token (因为没发 token 时, 团队和用户都更难被起诉), 也可能让团队选择"永远不发, 只做服务"。两种情况都不能排除。
