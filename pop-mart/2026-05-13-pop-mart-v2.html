用户最想知道的三件事

  1. 段永平不是回避了 IP 枯竭风险，而是承认它、不下注它。他公开说过"我依然无法理解人们为什么会需要这个东西，万一过两年大家都不要了呢？"——而且他从没收回这句话。他列的四大护城河里没有"IP 创新能力"这一条，他赌的是王宁挑 IP 的眼光和签约能力，不是赌 Labubu 永远不衰。
  2. 他破例了自己的「看不懂不投资」红线。2025 年 8 月到 2026 年 1 月他反复说"看不懂"，3 月看完年报 + 4 月 4 日去 Westfield 商场实地探店两趟后翻盘。原话："我理解王宁不是因为我的投资，而是因为我曾经是个'企业家'，我能看懂他有多厉害。"——他从"看懂生意"转到了"看懂人"，本质上松动了自己的能力圈原则。

02 — What He Actually Saw

他看到的是王宁，不是 Labubu

这是 v2 报告最关键的发现。用户的核心疑问是"段永平怎么规避了 IP 枯竭风险？"——答案是他没有规避。他公开承认这个风险一直存在，并选择把赌注下在"王宁这个人"身上而不是"Labubu 这个产品"身上。

段永平列的「四大壁垒」 · 三项半都关于人

关键发现：四条护城河里没有任何一条说"Labubu 会一直火"或"IP 创新能力强"。段永平知道他自己不懂 IP 周期——所以他干脆不下注 IP 本身。

"我理解王宁不是因为我的投资，而是因为我曾经是个'企业家'，我能看懂他有多厉害。他还那么年轻，他还能至少好好干 25 年以上吧。这个复利是吓人的。" 段永平 · 2026-05-07 雪球清仓神华当天

关键算术：25 年企业家复利

这句话的隐藏算式是惊人的：

  • 王宁现年 37 岁，假设干到 62 岁退休 → 还有 25 年
  • 段永平现年 65 岁，37 + 25 = 62 岁——刚好是段永平今天的年纪
  • 这不是巧合——段永平在赌"一个还有完整职业生涯的实业型 CEO + 25 年复利时间"
  • 对比：苹果 Tim Cook 接任时 50 岁、茅台老团队都是 60+ 岁——王宁是段永平见过的"创始人复利时间最长"的标的

段永平如何应对"IP 枯竭"风险？四层回应

第 1 层 · 公开承认风险

  • 2026 年 1 月（已研究 3 个月后）仍说："我依然无法理解人们为什么会需要这个东西，万一过两年大家都不要了呢？"
  • 这句话他至今没有彻底收回

第 2 层 · 主动回避用"IP 持续"作理由

  • 从未说过"我相信泡泡玛特能持续出新 IP"
  • 赌的是王宁挑 IP 的眼光（签约能力），不是赌 IP 本身
  • 这是一个微妙但关键的区分

第 3 层 · 用"模仿者复制不了"为护城河

  • 原话："已经有很多人砸了很多钱、非常努力地去复制泡泡玛特，花了同样的资本金，没有达到泡泡玛特十分之一的市场效果。"
  • 模仿者不是未来才出现，是早就出现了——已经证明复制不了

第 4 层 · 留 5-10 年观察窗口

  • "先看个五年十年吧"——不是笃定 25 年
  • 清醒承认自己不能 100% 看穿
  • 用时间检验王宁能否把"王宁的公司"变成"没有王宁也能跑的公司"

他真正看到的、普通投资者没看到的

来自段永平的实业经验

"我曾经是个'企业家'——做过 OPPO、vivo、步步高、小霸王。我知道一个中国品牌做到让外国人因为喜欢而买，有多难。OPPO/vivo 出海靠的是性价比 + 渠道，没真正做到。王宁做到了。这是他用 30 年实业经验换来的能力圈——不是普通投资者的能力圈。"

段永平在王宁身上看到了三件他自己曾经想做但没做到极致的事：

  1. 没有性价比叙事，靠纯审美让全世界买单——OPPO/vivo 出海靠的是性价比 + 渠道，王宁是第一个让外国人"因为喜欢而买"的中国消费品
  2. 37 岁还有 25 年职业生涯——这是企业家复利，不是 ROE 复利
  3. 暴赚中能看到危机的清醒——王宁 2025 年身家 USD 275 亿同时说"刚学会开车被拉去开 F1，组织管理打 70 分"——这种创始人不会被胜利冲昏头
03 — Wang Ning's IP Machine

王宁的 IP 发现机器：拆解

如果段永平赌的是王宁的"签约能力"而不是"Labubu 的持续力"，那这台机器到底是不是真的？它能不能在王宁不在 CEO 位置上的时候继续转？王宁本人公开给过最直接的答案——而且不止一次。

王宁本人原话（按时间排序）

"我们的 IP 孵化过程并不是基于我个人的喜好来决定的，而是有一个相对客观的评估标准和资源分配机制。这种资源分配机制是基于数据和市场反馈的，而不是基于个人主观判断。即使有一天我不再担任 CEO，这个体系仍然能够继续运行并孵化出新的 IP。" 王宁 · 2024-07 红杉爱生活访谈
"我们比迪士尼快。我们像抖音，尽量把资源切碎，我们看销售的表现去决定对一个 IP 的投入，以最快速度发现这个 IP 是不是值得投入更多资源，或者把它降权，所以我们每年营收最高的 IP 都在变化。" 王宁 · 2025 证券时报访谈
"做这个时代的"唱片"公司。就像唱片公司组织团队，帮歌手制作音乐、拍 MV、排演唱会，让歌手以一个人无法达到的速度出专辑、开演唱会。今天一个一个玩具 IP 也从泡泡玛特的娱乐工厂走出。" 王宁 · 2021《财经》专访（关键比喻）
"去年开始，我们希望不是成为中国的迪士尼，而是成为世界的泡泡玛特。平台孵化 IP 的方式，更能适应这个时代。" 王宁 · 2025-07 新华网

历史规律 · 反直觉发现

规律 1 · 30 年只需 2-3 个新爆款

  • Sanrio 65 年 = Hello Kitty + Cinnamoroll + Kuromi（平均 20 年才一个）
  • Bandai 75 年 = Gundam + Tamagotchi + Digimon（同样 20 年级别）
  • "每年出爆款"不是 IP 公司的运作模式

规律 2 · 老 IP 衰减是缓慢的

  • Hello Kitty 2002 年就丢角色榜第一，但靠"长尾 + 海外授权"活了 20 年
  • Digimon 2005-2007 销售跌 88%，但 IP 至今活着
  • "断崖式归零"是 Funko 模式（无 IP 矩阵），不是 Sanrio 模式

规律 3 · 低谷期 5-10 年是常态

  • Sanrio：7 年（两次）
  • Disney：18 年（1966-1984）
  • Bandai：8 年（1997-2005）
  • 股价跌 50-95% 是入场费

规律 4 · IP 创造是系统能力

  • Sanrio 1986 年起搞角色榜年度投票 = A/B 测试机制
  • Bandai 官方"IP 轴心战略" = 工业化流程
  • 不靠某个天才设计师的运气，靠流程 + 数据
历史一句话答案

"IP 公司没有下一个爆款不会死，但会经历 5-10 年（极端到 18 年）的低谷期，市值蒸发 50-95%。能活下来的关键不是「再出一个爆款」，而是三件事：① 老 IP 的授权金现金流够撑公司开支；② IP 矩阵已经有 3-5 个候补在孵化（哪怕不火）；③ 创始人/管理层愿意熬 7-15 年慢慢转型。Sanrio、Bandai、Disney 三家全部活下来，活下来后股价都涨了 10-50 倍。"

06 — Three Investors · Three Paths

三种投资者，三条路

v1 报告给的是"等中报 / 分批建仓 / 现价重仓"三条仓位路径——基于风险耐受度。v2 给的是基于投资认知的三条路径——你应该用段永平的方式，还是用普通投资者的方式？还是干脆不参与？

Three Investor Profiles

你属于哪种投资者？

段永平的方式有效，但前提是你具备段永平的认知——30 年实业经验、对创始人的判断力、5-10 年熬底的耐心。如果不具备，盲目模仿可能更危险。下面三类投资者，对应三条不同的路。

PROFILE A · 段永平式

赌人不赌产品

你能从王宁身上看到自己想做但没做到的事。愿意接受 5-10 年低谷期 + 50-90% 回撤。仓位 5% 上限，用卖 PUT（如有期权账户）或分批正股建仓。

PROFILE B · 量化派

盯 7 个信号买卖

不评判王宁这个人，纯粹按数据决策。海外增速 / 存货 / 减持 / FCF 7 个信号每月更新，2/7 翻负减仓，4/7 翻正加仓。仓位上限 5%。

PROFILE C · 静观派

等 DCF 安全边际

不下注王宁，不下注海外故事。等股价跌到中性 DCF 内在价值（HKD 65.7）±20% 区间再考虑。Pop Mart 不死，时间在你这边——错过这次也有下次。

段永平式（Profile A）的真实成本

必备前提

  • 实业经验：能从王宁的访谈/年报里读出他的清醒度，普通财务分析师看不出来
  • 5-10 年时间：Sanrio 复活花了 22 年；段永平自己说"先看个 5-10 年"
  • 能扛 50-90% 回撤：历史先例如此
  • 愿意承认看不懂 IP 周期，把不确定性留给王宁去解

真实代价

  • 5-10 年资金机会成本
  • 心理折磨：每次财报不及预期股价腰斩
  • 如果龙家升不续约 → 下跌可能不止 -90%
  • 段永平的边际成本是 5% 仓位，对他不致命；对普通投资者 5% 可能就是退休金

Profile B（量化派）的操作清单

不需要"看懂王宁"，按机械规则交易：

  1. 初始仓位：1% 试水，等满足 4/7 信号翻正再加到 3%，全满才到 5% 上限
  2. 每月更新 7 个信号（Google Trends / 闲鱼价 / Bloomberg 季度数据 / HKEx 大股东持股变动 / 存货周转 / 派息政策 / 海外门店净增）
  3. 2/7 翻负 → 减仓 1/3；4/7 翻负 → 清仓
  4. HKD 130-140 加仓 1/2，HKD 100 加仓全部剩余
  5. HKD 250 卖 1/3，HKD 320 卖 1/3，剩 1/3 长持

Profile C（静观派）的等待逻辑

为什么不动手是合理的

"Pop Mart 不会死，时间在你这边。Sanrio 2003 年股价 ¥152 是历史最低点，从这里涨 57 倍到 ¥8,685——但如果你 1999 年高点 ¥3000+ 买入，要等 22 年才解套。如果你能等 DCF 中性内在价值（HKD 65.7），即使最终只回到中性情景终值 HKD 70，5 年收益就是 0%；如果回到现价 HKD 162.9，5 年 +148%；如果回到 ATH HKD 339.8，5 年 +418%。错过一次没关系——这种公司一辈子至少出现 3 次极值机会，等下次就是了。"

不论选哪条，避开的 3 个坑

  • 不要把"段永平买了我也买"当核心理由——他下注的是自己 30 年实业经验形成的能力圈，你的能力圈不同。模仿别人是最弱的投资逻辑。
  • 不要忽略他没有规避的风险——他公开说过"万一过两年大家都不要了呢"，至今没收回。如果他都承认这是风险，普通投资者更不该假设它不存在。
  • 不要把 25 年企业家复利当确定性——段永平自己说"先看个 5-10 年"，他给自己留了观察窗口。如果你重仓的话，至少给自己留同样的观察窗口。

最终一句话

"普通投资者看的是 Labubu、IP、毛利率、Z 世代消费——这些都是结果。段永平看的是这一切结果背后的那个人。这是他用 30 年实业经验换来的能力圈，不是普通投资者的能力圈。这个判断如果错了，错的代价不仅是这笔投资，而是他多年来在雪球公开教散户的核心信条——'看不懂不投资'。"
