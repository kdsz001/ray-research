## 核心结论

- **2025 是「神话年」，2026 是「证明年」** — 营收 ¥371 亿 / 净利 ¥130 亿 / +185% 是真的，但管理层把 2026 定为「进站维修年，目标 +20%」，从 +185% 降到 +20% 是断崖式回落，市场用 -52.7% 股价跌幅提前 price in
- **估值便宜得离谱 vs 单 IP 风险真实存在** — TTM PE 15.3x / PEG 0.24 / ROE 78%（全球可比最低），但 Labubu 单 IP 占 38%，Google Trends 已跌 92%、隐藏款二手价跌 98.96%，Funko 化路径所有信号都已出现
- **未来 12 个月只回答一个问题：「下一个 Labubu」能否出现** — 能 → 走 Sanrio 路径，$27.6B 市值远未到顶；不能 → 走 Funko 路径，现价仍有 30-50% 下行空间
- **当前位置是「中等性价比逆向博弈」** — 三条路径：等中报（低风险）/ 分批建仓（中风险）/ 现价重仓博反弹（高风险）
